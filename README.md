# Abigail & Ricardo 💕

Página web interactiva creada como detalle especial para celebrar **1 año y 11 meses** juntos.

El sitio incluye una pantalla de desbloqueo personalizada, una ruleta de planes, selección de hora y un mensaje final romántico para Abigail.

## ✨ Características

- Pantalla inicial con clave personalizada.
- Desbloqueo con los nombres: `Abigail`, `abigail` o `abi`.
- Animación estilo “matrix” como fondo.
- Ruleta interactiva con planes para una cita.
- Selector de hora.
- Mensaje final personalizado.
- Diseño responsive para abrir desde celular o computadora.

## 🚀 Publicación con GitHub Pages

1. Crear un repositorio público en GitHub.
2. Subir el archivo `index.html`, este `README.md` y la carpeta `assets`.
3. Ir a **Settings → Pages**.
4. En **Build and deployment**, seleccionar:
   - **Source:** Deploy from a branch
   - **Branch:** main
   - **Folder:** /root
5. Guardar los cambios.
6. Esperar unos minutos hasta que GitHub genere el enlace.

El enlace tendrá un formato similar a:

```text
https://TU-USUARIO.github.io/NOMBRE-DEL-REPOSITORIO/
```
